How to use Swish Fish

The GitHub repository in the sidebar contains all necessary files for SwishFish. There are 2 subfolders required to set up:

  • The Controls folder contains the files required to set up the logistic regression model for gesture controls.
  • The Game folder contains the files required to set up the pygame based simulation

First, download all files locally. With Python 3.5, install the Pipenv virtual environments in each of these subfolders. In two separate terminal windows, in each virtual environment, navigate to the parent folder of the controls and game subfolders.

With the terminal in the controls virtual environment, run the command python SwishFish-controls/Predict.py -h to see the command line arguments required to set up the gesture controls.

Once this is set up, with the game virtual environment terminal, just run python SwishFish-game/main.py to set up the game (and click on the window that pops up to initialise the logistic regression training - this is a side-effect of the piping system)

The reason for being in the parent folder to controls and game is due to using UNIX pipes (through python's os module) to send gesture commands between the predictor and the game.
